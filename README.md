# Title // ENGLISH
Apple Click Game

# Description
Game for browsers (tested in google chrome), it has three levels: easy, normal and hard. The Objective is to click apples to increase your score.

Technologies used:

- HTML / CSS
- Javascript
- Bootstrap 4
- jQuery 3.3.1
